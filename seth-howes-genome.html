Seth Howes 家庭基因组测序案例

一个英国软件工程师在自家厨房桌面完成全基因组测序：从口腔拭子采样、DNA 提取、文库构建、MinION 测序，到 Dorado basecalling 与 minimap2 比对，全程离线完成。

72h从样本到 aligned BAM 的完整闭环
~30Gb纳米孔测序数据产出
~10x全基因组平均覆盖度
$1,100单次耗材成本估算

Case Overview

为什么这个案例值得放进项目集

基因组学工具民主化

案例展示了 MinION 这类便携式测序仪如何把过去属于大型中心实验室的能力，下沉到个人工作台。Seth 没有把 DNA 寄给消费级检测公司，也没有把数据上传云端，而是完成了本地实验与本地计算。

GLI 闭环的真人样本

从 AI 辅助设计 Panel，到手动完成湿实验，再到测序、basecalling、比对和后续变异分析，这个案例对应 Design、Build、Test、Learn 四个环节，是自动化实验系统的手动版原型。

Technical Stack

核心系统拆解

硬件

Oxford Nanopore MinION Mk1D、R10.4.1 flow cell、加热块、微量离心机、涡旋混合器、移液器、DIY 磁力架和基础湿实验耗材。

MinION · Flow Cell · Pipette · DIY Lab

湿实验

口腔拭子采集颊细胞，Monarch gDNA 提取，SQK-LSK114 连接法建库，Flow Cell priming 与上样，48 小时连续测序。

DNA Extraction · Library Prep · Sequencing

干实验

MinKNOW 采集信号，Dorado HAC 实时 basecalling，SUP 后处理提升精度，minimap2 与 samtools 生成 aligned BAM 并完成质控。

Dorado · minimap2 · samtools · BAM

Workflow

从样本到基因组文件

01 DESIGN

设计测序策略

用 AI 辅助整理基因知识、设计目标 Panel，并决定全基因组浅层或目标区域深度测序策略。

02 BUILD

提取与建库

从口腔拭子中提取 gDNA，完成末端修复、磁珠纯化和 adapter 连接，得到可上机文库。

03 TEST

MinION 测序

加载 flow cell，运行 48 小时，持续监控 pore occupancy、read length 和 translocation speed。

04 COMPUTE

本地计算

用 Dorado 做 basecalling，再通过 minimap2、samtools、mosdepth 完成比对、排序、索引和覆盖度分析。

05 LEARN

反馈优化

根据覆盖度、失败点和变异分析结果，优化下一轮实验条件、Panel 设计和自动化策略。

Lessons Learned

关键踩坑经验

实验侧风险

DNA 定量要前置；flow cell 最怕气泡；酶类试剂不能涡旋；Ligation Buffer 粘稠，需要慢速吸打混匀；第二次纯化必须使用 Long Fragment Buffer，不能用乙醇替代。

工程侧启示

真正困难的不是单个步骤，而是每一步都要稳定可复现。移液器精度、气泡控制、磁珠干燥时间、计算硬件加速，都会影响整个闭环的可靠性和成本。

GLI Mapping

对自动化实验的启示

Seth 做的是 Design-Build-Test-Learn 闭环的“手动版”。GLI 的价值，是把人的移液、监控、分析和迭代判断，逐步交给自动化工作站与 AI Agent。

Design

LLM 整合 UCSC、Ensembl、OMIM、CPIC 等知识，辅助生成 BED 文件和实验方案。

Build / Test

人工湿实验可映射为移液机器人、样品处理设备和测序仪的自动化执行与监控。

Learn

AI 分析测序结果、定位失败原因、提出下一轮实验参数，形成反馈优化闭环。
